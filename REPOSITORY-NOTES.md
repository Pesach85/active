# Repository Notes

This repository is prepared for local versioning and controlled sharing.

## Sensitive data policy
- Runtime logs are ignored.
- Generated CSV reports are ignored.
- KB runtime journal is ignored.
- Built EXE binaries are ignored.

## Before publishing to a remote
1. Review git status.
2. Run a secrets scan on staged content.
3. Confirm no personal paths, credentials, tokens, or host-specific data are present.
